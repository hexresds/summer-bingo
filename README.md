# Summer Bingo

Easy game to track the activities you have completed in order to have an unforgettable summer.

## Getting started

1. Clone the repo

```
git clone https://github.com/hexresds/summer-bingo
cd summer-bingo
```

2. Install dependencies

```
npm install
```

3. Run the app

```
npm start
```

## Built with

- React (creat react app)
- CSS
- Javascript
